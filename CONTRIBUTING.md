# Contributing to Symptothermie App

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/symptothermie-app.git`
3. Create a branch: `git checkout -b feature/your-feature-name`

## Development Setup

See the main README.md for installation instructions.

## Pull Request Process

1. Ensure your code follows the existing style
2. Add tests for new features
3. Update documentation as needed
4. Submit your pull request with a clear description

## Code of Conduct

Be respectful and constructive in all interactions.
